Favorite Project

My favorite project from Web & App Development this year is the Pacific Trails Resort project. This was a solo project, and we had to create a website with three separate pages to view, each being designed with CSS and having the ability to navigate to the other two pages. I enjoyed this project just because it was kind of like a glimpse of what we would really be creating as Web Designers, and just being able to see what goes into a page this developed and making it yourself. It was definitely hard, it used almost everything we learned and a bit more, but it was also fun at the same time, almost like a puzzle figuring out what does what and what goes where.
